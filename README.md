# Общее описание решения

**Создал файлы с функциями для вычисления площадей и периметров геометрических фигур:**
- `circle.py` -> `area`, `perimeter`
- `rectangle.py` -> `area`, `perimeter`
- `square.py` -> `area`, `perimeter`
- `triangle.py` -> `area`, `perimeter`

# Описание функций 
## Circle
### Area
> **Принимает радиус круга (десятичное число - r). Возвращает площадь круга (десятичное число - r² умноженное на математическую константу π).**
#### Пример использования
- `circle.area(10)` -> `314.1592653589793`
### Perimeter
> **Принимает радиус круга (десятичное число - r). Возвращает периметр круга (десятичное число - 2r умноженное на математическую константу π).**
#### Пример использования
- `circle.perimeter(10)` -> `62.83185307179586`


## Rectangle
### Area
> **Принимает сопряженные стороны прямоугольника (десятичные числа - a и b). Возвращает площадь прямоугольника (десятичное число - произведение a и b).**
#### Пример использования
- `rectangle.area(10, 1)` -> `10`
### Perimeter
> **Принимает сопряженные стороны прямоугольника (десятичные числа - a и b). Возвращает периметр прямоугольника (десятичное число - сумма a и b умноженная на 2).**
#### Пример использования
- `rectangle.perimeter(10, 1)` -> `22`


## Square
### Area
> **Принимает длину стороны квадрата (десятичное число - a). Возвращает площадь квадрата (десятичное число - произведение a на a).**
#### Пример использования
- `square.area(10)` -> `100`
### Perimeter
> **Принимает длину стороны квадрата (десятичное число - a). Возвращает периметр квадрата (десятичное число - a умноженное на 4).**
#### Пример использования
- `square.perimeter(10)` -> `40`


## Triangle
### Area
> **Принимает длину стороны и длину высоту треугольника перпендикулярную этой стороне (десятичные числа - a и height). Возвращает площадь треугольника (десятичное число - произведение a на height деленное на 2).**
#### Пример использования
- `triangle.area(10, 2)` -> `10`
### Perimeter
> **Принимает длину стороны треугольника (десятичные числа - a, b и с). Возвращает периметр трегольника (десятичное число - сумма a, b, и с).**
#### Пример использования
- `triangle.perimeter(10, 5, 3)` -> `18`
